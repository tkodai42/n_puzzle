# n_puzzle

SYNOPSIS

     n_puzzle [-...] [file]

DESCRIPTION

     The options are as follows:

     -f1   The Manhattan-distance

     -f2   Improved Manhattan-distance

     -f3   Corrent number of pieces

     -f4   Euclidean-distance

     -f5   Search bogo => ...random

     -f6   Uniform Cost

     -f7   greedy search

     -f8   Original

     -f9   greedy + Improved Manhattan-distance

     -g    debug

     -h    HELP

     -p    play n_puzzle!!!

     -v    visualize
